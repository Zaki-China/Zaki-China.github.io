HEXO+Github,搭建属于自己的博客

注意：本教程只针对windows用户

一、本地搭建hexo

1、安装GitBash

官网下载，并安装

2、安装Node.js

官网下载：https://nodejs.org/en/
安装时保持默认配置。

3、安装hexo

在任意位置，右键打开GitBash，利用npm命令进行安装
npm install -g hexo

4、创建hexo文件夹

安装完成后，在电脑中新建一个自己喜欢的文件夹（如I:\hexo）,然后在文件夹内右键选着GitBash，在GitBash工具下使用命令安装配置hexo。
hexo init
过程可能有点慢，需要下载一些东西，需要赖心等待。

5、安装依赖包
